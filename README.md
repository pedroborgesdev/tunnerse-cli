# Tunnerse CLI

**Expose your on-premises application to the entire internet via reverse tunnels!**

---

## Installation
1. See the releases and download the most current version of the binary equivalent to your operating system.
2. Place the binary in a folder and register that folder in your system PATH. If you are using Linux, move the binary to the executable binaries folder.

---

## Usage
Expose quickly
```bash
$ tunnerse <tunnel_name> <local_port>

# Example
$ tunnerse tunnel-example 8080
```
